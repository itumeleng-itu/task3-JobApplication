What is a web app:
software that runs on a remote server and is accessed over the internet through
 a web browser on any device, rather than being installed on the device itself

Purpose of a web app:
The primary purpose of a web application is to allow users to perform tasks and
 access services through a web browser   
1. Provides Functionality (allows user interaction)
2. Delivers Remote Services (delivers its features over the internet through a web browser)
3. Enables Interaction and Data Exchange (send and receive data between their browser and the server, engaging in actions like logging in)

Benefits:
1. Accessibility
2. Cross-Platform Compatibility
3. Scalability
4. Easy Updates

Examples: 

1. Easy Updates
2. Document editors
3. Email services
